# Configuração de ambiente

### Clonar o projeto
https://github.com/Tomazzini/proffys.git

### Run Api.
Go to server folder.
cd proffy/server

Install Dependencies.
$ yarn install

Run Aplication
$ yarn start

knex create migration
$ yarn knex:migrate

knex create rollback
$ yarn knex:rollback

### Run Projeto Web.
Go to web folder.
$ cd proffy/web.

Install Dependencies.
$ yarn install.

Run Aplication
$ yarn start

### Run Projeto Mobile
Go to mobile folder
$ cd proffy/mobile

Install Dependencies
$ yarn install

Run Aplication
$ yarn start

### Tecnologias
Typescript.
React Native.
Expo.
Express.
